## what this was

a github repo kept **public**, updated before every practical exam, containing working lab programs — so that anyone in the batch could open it, copy the code, change variable names, and submit.

no fancy setup. no install. just:

```
open github → find the program → copy → paste → rename variables → submit → poker face
```

ran for **730 days**. every semester. zero issues.
